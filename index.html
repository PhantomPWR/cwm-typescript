Typescript

Getting Started

Install Typescript
npm i -g typescript
Confirm Installation
tsc -v
Create Configuration File
tsc --init
Compile Typescript
tsc filename.ts

NOTE: If rootDir is set in tsconfig.json, only run tsc.

Debugging
To debug Typescript in the browser, set sourceMap to true in tsconfig.json.
  1. Open a .ts file and add a breakpoint
  2. Click on "Run and Debug"
  3. Click on "create launch.json file"
  4. From the dropdown, select "Node.js"
  5. Under "configurations", add "preLaunchTask": "tsc: build - tsconfig.json"

Types

Typescript automatically assigns types to variables, e.g:

let sales: number = 123_456_789;

can be written as

let sales = 123_456_789;
